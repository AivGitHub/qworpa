## Installation

1. `python -m venv .venv`.
2. `source .venv/bin/activate`.
3. `pip install -r requirements.txt`.

## Development

1. Export variables from `.env.template` with corresponding values.
2. `python manage.py runserver` or `bash entrypoint.sh`.
